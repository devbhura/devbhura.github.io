Introduction

The report below details the process of building several controllers, such as PD and PID controllers for 2-DOF systems. The 2-DOF system consists of two circular plates connected by a flexible rod that acts as a torsional spring, as shown below in figure 1.

Torque τ is applied to the lower plate which moves the plate I1. The plate I1 moves an angle θ1, which is directly controlled by the controller. I1 is also connected to the plate I2 through a flexible rod. The motion in I1 causes motion in I2. This motion is captured by θ2. The objective in this report is to show various controllers for I1 and I2.

To identify the 2-DOF system above, the following assumptions have been made: 1. The system is linear; 2. There is no stiction; 3. The hardware can be modeled as a simple gain instead of a complex function; 4. There is mo delay in the system; 5. The system is continuous.

Using these assumptions, the system is said to be governed by the differential equations of motion given below:

$$I_1\ddot{\theta_1} + (c_1 + c_2) \dot{\theta_1} + (k_1+ k_2)\theta_1 - c_2\dot{\theta_2} - k_2\theta_2 = \tau(t)$$

$$I_2\ddot{\theta_2} + (c_2 + c_3) \dot{\theta_2} + (k_2+ k_3)\theta_2 - c_2\dot{\theta_1} - k_2\theta_1 = 0$$

Taking Laplace transform and rearranging:

X1(s) + [−c2s−k2]X2(s) = τ(s)

X2(s) + [−c2s−k2]X1(s) = 0

Written in matrix form: D(s)X(s) = F(s) $$D(s) = \begin{bmatrix} I_1s^2 + (c_1 + c_2)s + (k_1 + k_2) & -c_2s - k_2\\ -c_2s - k_2 & I_2s^2 + (c_2 + c_3)s + (k_2 + k_3)\\ \end{bmatrix}; X(s) = \begin{bmatrix} X_1(s)\\ X_2(s)\\ \end{bmatrix}; F(s) = \begin{bmatrix} \tau(s)\\ 0\\ \end{bmatrix}$$

Now, X(s) = D(s)−1F(s)

Simplifying this matrix operation, and assuming τ(s) = KHWR(s) we can say that: $$\frac{X_1(s)}{R(s)} = G_{col} = K_{HW} \frac{I_2s^2 + (c_2 + c_3)s + (k_2 + k_3)}{det(D(s))}$$

This transfer function Gcol defines the motion of the collocated system I1. Collocated system is controlled directly by the actuator.

Similarly, $$\frac{X_2(s)}{R(s)} = G_{ncol} =K_{HW} \frac{c_2s + k_2}{det(D(s))}$$

The above equation defines the motion of the non-collocated plant, that is I2. This plant is called non-collocated as it is not directly controlled by the actuators, but rather is influenced by the motion of I1.

The ratio between Gcol and Gncol is G21: $G_{21} = \frac{G_{col}}{G_{ncol}}$

System Identification

To identify the system parameters, frequency domain analysis was conducted. The frequency response of a system was obtained by giving the system a sinesweep and capturing its response, and also by giving it a single sinusoidal input and recording its output. These results from these two methods are shown below.

From these graphs, we can get ω1, ω2, ωz, ζ1, ζ2, ζz using the Half-Power method. These parameters define the system in terms of the model parameters. The Gcol and Gncol defined in equation (6) and (7) can also be defined by the following equations:

$$G_{col} = K' \frac{s^2 + 2\omega_z\zeta_zs + \omega_z^2}{(s^2 + 2\omega_1\zeta_1s + \omega_1^2)(s^2 + 2\omega_2\zeta_2s + \omega_2^2)}$$

$$G_{ncol} = K' \frac{\omega_z^2}{(s^2 + 2\omega_1\zeta_1s + \omega_1^2)(s^2 + 2\omega_2\zeta_2s + \omega_2^2)}$$

$$G_{21} = K' \frac{\omega_z^2}{s^2 + 2\omega_z\zeta_zs + \omega_z^2}$$

K’ can be found by normalizing the steady state amplitude of the individual sine by the input amplitude.

Analysis & Conclusion

Figure 3 and 4 illustrate the frequency domain response of the 2-DOF system. Figure 3 shows the response of the collocated disc. There are two peaks and one trough in the response. This is different from the 1-DOF system response which had only one peak. The non-collocated system response show in figure 4 shows that there is a peak at two frequencies, although no trough. Analysing the figures 3b and 4b, nature of modes of vibration can be seen. At the first resonant frequency ω1, both discs will rotate in the same direction as per the phase plots. At omega2 however, the phase plot suggest that the direction of motion will be opposite to each other.

The results from the sinesweep data as well as from the individual sine experiment are very close, although the damping for the non-collocated response is marginally lower (as shown in table 1). Using the model with the least damping, the model is created. Figure 5 and 6 show the open loop results of the experiment along with the analytical model response to a step input of 500 counts. Figure 5 shows that the model has underestimated the damping completely, while figure 6 shows that although the model is over-damped for the first peak, it is under-damped compared to the entire experimental response.

It is better to underestimate damping rather than overestimate it. The controller can bring the system to steady state, by compensating for less damping as predicted by model which this is safer than the controller assuming that system behaves better than it actually is and not bringing the system to steady state. From figures 5 and 6, we can see that the analytical model either overestimates damping or is very close to the experimental response. Therefore the model is good enough for a controller design. The controller will work for the worst case scenario.

Feedback Control

A controller is used to drive the system to the desired output. One of the hallmarks of a controller is using a negative feedback loop. Feedback controller means that the output of the system is fed in as an input to the controller. Negative feedback means that this feedback is subtracted from the input to give us the error. The controller is designed such that it minimizes this error to drive the system to the desired output. In a 2-DOF system controller, different controller designs are used to control different things.

Collocated controllers are used to control the disc connected to the actuator directly. It is the most effective controller. The block diagram below shows how the Collocated controller is set up. It aims to control x1 by controlling the response of the collocated plant Gcol which is defined in equation 8. In the block diagrams, U(s) represents the control effort, X1(s) the position θ1 of directly controlled disc while X2(s) the position θ2 of the indirectly controlled disc.

Non-Collocated Controllers are designed to control the second disk that is connected to the first disk by a flexible rod. The block diagram below shows how the Non-Collocated controller is set up. It aims to control x2 by controlling the response of the collocated plant Gncol which is defined in equation 9. Gncol = Gcol • G21 in the figure below.

Mixed Controllers are designed to control the velocity of the collocated plant and the position of the non-collocated plant, as shown in the figure below. It takes x1 and does a feedback derivative control on that while it takes x2 and controls its position using proportional control. This is a hybrid controller meant to control both the disks.

These 3 control objected are achieved in various ways.

Design approach

The aim was to minimize the maximum overshoot and the settling time of the response, the control effort and the steady state error. This was done through PD and PID controllers, in their cascade and velocity feedback architectures (block diagram shown for each individual controller design). These controllers were designed using sisotool. The root locus was changed to give the desired output. The figure 10 a shows root locus of the PD series controlled system. To increase the design space, the real zero was moved sufficiently. The closed loop poles shown in solid pink are moved around to give the desired results, either to reduce maximum overshoot, settling time, control effort or steady state error.

PD Rate Feedback

The rate feedback PD Controller is used to control the position of disc 1. The control architecture uses derivative control KD on the velocity of the collocated plant, and a proportional controller KP for the position of the collocated plant, as shown in the block diagram below.

The root-locus design in sisotool is shown below in figure 12. The root locus shows that there are 2 complex closed loop poles. Moving them gives the desired results. However increasing design space by moving the open loop zeros. The tiny branches seen in the root locus for looptransfer C2 do not move when the open loop zero is moved. This limits the design space. The complex poles have a small range where it is stable. If those poles are too close to the imaginary axis, the root locus in C1 becomes unstable. Therefore only one design in the architecture was possible.

The figure 13 shows the experimental and analytical response for Kp = 0.56, KD = 0.04 designed in figure 12. The analytical response is close to the experimental response. The design has almost no overshoot. It settles in around 0.5 seconds, which is quite fast. It also has a small steady state error at 29 counts. The analytical steady state is slightly higher than the experimental one. This is due to viscous friction in the real system which makes the system settle to a lower steady state. The steady state error cannot be zero as PD controller cannot achieve it.

This design is still quite good, although the only one for the rate feedback. The control effort, shown in figure 14, settles to a sufficiently low value, although it starts at quite a high value. This is another limitation of the velocity feedback design, as further increase in KP for a better controller would cause a higher control effort which would not be implementable.

Conclusions

The PD series controller is the most preferred form of control for controlling disc 1. It gives very low steady state error even without integral control. The overshoot and settling time were minimal while the control effort was implementable. There is also ample design space to work with to get the desirable outcome. Non-collocated controllers have very limited design space. Therefore to control the second disc, mixed control would be preferred. It has a greater design space and can be implemented.
